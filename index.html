Laços de Repetição

Os laços de repetição são uma forma de repetir um trecho de código várias vezes de maneira rápida e prática. Eles permitem executar uma ação repetidamente sem precisar escrever o mesmo comando diversas vezes. Um bom exemplo seria escrever manualmente os números de 1 a 1000 — isso daria muito trabalho! No entanto, em JavaScript, com poucas linhas de código, conseguimos imprimir toda essa contagem em poucos segundos.

Exemplo:
for (let i = 0; i < 1000; i++) {
  console.log(i);
}

Existem várias formas de criar laços, mas todos têm o mesmo objetivo: repetir uma ação várias vezes. Em alguns casos, será mais fácil usar um tipo específico de laço, dependendo da situação, mesmo que todos cumpram a mesma função básica.

1 - For

O laço for cria um loop com três expressões opcionais.

Exemplo:
for (expr1; expr2; expr3) {
  // código a ser executado
}

A primeira expressão (expr1) é executada uma vez antes do bloco de código. A segunda expressão (expr2) define a condição que será verificada para continuar executando o loop. Por fim, a terceira expressão (expr3) é executada após cada iteração do bloco de código.

for (let i = 0; i <= 5; i++) {
  text += "O número é " + i;
}

A variável i é definida antes do início do loop e começa com o valor 0, funcionando como um contador. A expressão expr2 define que o loop continuará enquanto i for menor ou igual a 5. Já a expressão expr3 (i++) incrementa o valor de i em 1 a cada iteração. Assim, o loop começa em 0 e termina quando i chega a 5.

2 - While

O loop while executa um bloco de código enquanto uma condição especificada for verdadeira. Usando o mesmo exemplo da variável i, o código ficaria assim:

let i = 0;
while (i < 10) {
  text += "O número é " + i;
  i++;
}

Declaramos a variável i com valor inicial 0 antes do while. Enquanto i for menor que 10, o bloco de código será executado. É importante incluir o i++ dentro do loop, pois, caso contrário, i nunca aumentará e o loop será executado infinitamente.

3 - Do While

O do...while é semelhante ao while, mas com uma diferença importante: ele executa o bloco de código uma vez antes de verificar a condição. Depois disso, repete o loop enquanto a condição for verdadeira.

let i = 0;
do {
  text += "O número é " + i;
  i++;
} while (i < 10);
